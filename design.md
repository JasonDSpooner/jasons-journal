# Design Document

## UI/UX Specification

### Layout Structure
- **Header**: Logo, navigation, user avatar/login button
- **Main Content**: Dynamic based on route
- **Sidebar**: Dashboard navigation (when logged in)
- **Footer**: Minimal with links

### Pages
1. **Landing Page**: Hero, features, login CTA
2. **Dashboard**: User stats, recent journals, quick actions
3. **Skills Page**: Grid of skill cards with descriptions
4. **AI Tools**: Text/image generation interface
5. **Journal**: List and editor for MD entries
6. **Gallery**: Image viewer with grid layout

### Visual Design
- **Color Palette**: 
  - Primary: #2563EB (Blue)
  - Secondary: #10B981 (Emerald)
  - Background: #F8FAFC
  - Dark: #1E293B
- **Typography**: Inter font family
- **Spacing**: 8px base unit
- **Border Radius**: 8px default, 12px for cards

### Components
- Button (primary, secondary, ghost)
- Card (with hover states)
- Input fields
- Modal/Dialog
- Navigation tabs
- Avatar
- Toast notifications
